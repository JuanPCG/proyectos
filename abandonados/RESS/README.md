# RESS
Recortador de Enlaces Super Simple


# Instalacion
## Paso 1:
Descarga y modifica los archivos **index.php** y **db.sql** y **MODIFICA** la informacion que sea necesaria. Lo mas probable es que quieras cambiar el usuario y la contraseña.
## Paso 2: 
Pon el archivo PHP en cualquier parte de tu servidor web, la URL no importa, aunque si el proposito es recortar enlaces, recomiendo ponerlo en la raiz con un nombre corto, o hacer un _rewrite_ en el .htaccess para que rediriga tudominio.tld/123 a tudominio.tld/recortador.php?123
